# Nauka Python — 30-dniowy projekt

To repozytorium to mój 30-dniowy projekt nauki Pythona: codzienna nauka + regularne mini-projekty.

## Plan
- **Pon–Pt:** 30 minut ćwiczeń dziennie
- **Sobota:** mini-projekt (wrzucany do repo)
- **Niedziela:** wolne / lekka powtórka (opcjonalnie)

## Struktura repo
- `cwiczenia/` — ćwiczenia z dni powszednich (podział na tygodnie i dni)
- `mini_projekty/` — projekty sobotnie (każdy ma swój folder)

## Jak uruchomić pliki
W terminalu w folderze projektu:
```bash
python nazwa_pliku.py
